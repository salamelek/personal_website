Passwords

A password manager that uses hashing to generate a different password for each key pair.

I never actually used this, but it was a fun project. It uses a pair of secret and public key to generate the password for any account. The secret key is the keyring password, while the public key is the name of the application of the account. The program spews out a different password for every key pair with the help of hashing.

While the Python version worked fine, it was very inconvenient to use. That's why I made a web version and later also the browser plugin version (I didn't bother putting it on the web store though).

Submarine

My dream of creating an RC submarine from scratch with an arduino, some motors and lots of luck.

The plan is to have a submarine controlled from my pc on land via an RF24 module for arduino. I made a Python script that captures keyboard or joystick events and sends them to an arduino nano via USB that in turn transmits via RF24 to the submarine antenna, where a second arduino translates the packets and controls the submarine.

Like any other RC submarine, I can't just rely on wireless communication, since 2.4GhZ waves barely penetrate water. My plan is to tether the sub to a floating antenna that will get all the signals, while allowing the submarine all the wanted mobility.

Besides the basic movement functions I also want to add an FPV camera, so I can see live where it's going on my computer screen.

For now the project is stuck on the ballast design, but I think that eventually it will be done.
